## KaizHudda.com

### Overview

Here you will find the source files for my very own Digital Resume website - [kaizhudda.com](kaizhudda.com)

### Stack

Created with: 
* React.js 
* Redux 
* HTML5 
* SCSS 
* Jest/Enzyme
* Contentful (Headless CMS)
* Netlify (CI/CD pipelines)

### Run

```yarn start```

### Build

```yarn build```

### Test

```yarn test-coverage```
